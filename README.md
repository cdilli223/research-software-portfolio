# Software Contribution Portfolio

This repository contains non-confidential summaries of representative software projects I have worked on in research, teaching, course, and employer-owned contexts.

Most of the underlying repositories are not public because they include institutional, course, research-group, student, or employer-owned materials that I am not authorized to redistribute. For that reason, this repository does not reproduce private source code, student submissions, internal repository structure, private data, credentials, or confidential implementation details.

Instead, these summaries describe my individual contributions, the software problem each project addressed, implementation and validation approaches, and what I would improve in a future iteration.

## Project Summaries

### [Differentiable APIC-DFSPH Solver Blending](summaries/differentiable-apic-dfsph-solver-blending.md)
Numerical simulation, differentiable programming, particle-grid methods, learned solver coupling, gate visualizations, velocity statistics, and rollout comparisons.

### [Automated Simulation Grading and Validation Infrastructure](summaries/automated-simulation-grading-and-validation.md)
Python automation for validating simulation submissions, checking expected folder/file structure, comparing outputs against an oracle simulation, and classifying structural, runtime, timeout, malformed-output, and numerical-divergence failures.

### [Gravitational-Wave Numerical Modeling Workflow](summaries/gravitational-wave-numerical-modeling-workflow.md)
Astrophysics research computing with Python and Mathematica scripts for modeling compact-object gravitational-wave quantities, frequency thresholds, signal-strength estimates, and detectability behavior across size, orientation, and proximity assumptions.

### [Interactive Signed-Distance-Field Geometry and Visualization Tool](summaries/interactive-sdf-geometry-tool.md)
Computer graphics and geometry-processing tool for drawing primitives and freehand closed curves, computing signed distance fields, visualizing distance-based behavior, and debugging sign/aggregation behavior across multiple objects.

### [Educational Technology Platform Integration and Assessment Infrastructure](summaries/educational-technology-platform-integration.md)
Employer-owned platform implementation work involving Canvas, Learnosity, platform migration, assessment workflows, QA, documentation, user support, and cross-functional implementation planning.

## What This Portfolio Demonstrates

Across these projects, my work has focused on translating technical requirements into reliable computational workflows. The projects include scientific simulation, research modeling, grading infrastructure, graphics tools, and educational technology systems. Common themes include:

- Building reproducible numerical or technical workflows
- Designing validation logic and diagnostics
- Debugging structural and numerical failure modes
- Translating mathematical, physical, or instructional requirements into software behavior
- Documenting assumptions, limitations, and future improvements
- Respecting institutional ownership and confidentiality constraints
